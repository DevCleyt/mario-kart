# 🏁 Mario Kart Console Game (Node.js)

Projeto desenvolvido em **JavaScript (Node.js)** durante o **Bootcamp Node.js da DIO (Digital Innovation One)**.

Este mini game simula uma corrida inspirada no universo do Mario, utilizando conceitos fundamentais de programação e lógica aplicados no ambiente Node.js.

---

## 🚀 Sobre o Projeto

O jogo funciona diretamente no terminal e simula uma corrida com as seguintes regras:

* 🎮 Dois personagens são sorteados automaticamente
* 🏁 A corrida possui 5 rodadas
* 🎲 Cada rodada possui um tipo de desafio aleatório:

  * **RETA** → disputa de velocidade
  * **CURVA** → disputa de manobrabilidade
  * **CONFRONTO** → disputa de poder (com penalidade para o perdedor)
* 🏆 Ao final, o jogador com mais pontos vence

---

## 🧠 Conceitos Aplicados

Durante o desenvolvimento foram aplicados:

* Programação Orientada a Objetos (Classes e Instâncias)
* Estruturas condicionais (`if`)
* Laços de repetição (`for`)
* Funções
* Arrays
* Lógica de sorteio com `Math.random()`
* Organização de código
* Simulação de engine de jogo

## 💻 Como Executar

1. Instale o Node.js
2. Clone o repositório
3. No terminal, execute:

```bash
node index.js
```
